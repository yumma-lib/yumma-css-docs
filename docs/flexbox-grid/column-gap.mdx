Utilities to control the spacing between columns in a grid layout.

:::info Creating Grid Layouts with Column Gaps
Specify a container element and apply the following classes to achieve a precise grid layout with fine-grained column spacing:

- `cg-*`: Control column spacing.
- `rg-*`: Control row spacing.
- `gtc-*`: Specify grid columns.
- `gtr-*`: Specify grid rows.
- `d-g-*`: Designate as a grid container.
:::
