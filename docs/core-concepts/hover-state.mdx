# Hover State

Manages the appearance of elements when users hover over them, adding interactivity.

### Basic examples

#### Background Color

<CodePreview
  htmlContent={`
    <div class="bg-magenta bg-h-black h-6 p-1 rad-1 w-6"></div>
  `}
  isPadded = {true}
  isCentered = {true}
  isVersion = 'latest'
/>

```html
<div class="bg-h-black ..."></div>
```
