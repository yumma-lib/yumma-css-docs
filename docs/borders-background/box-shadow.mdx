# Box Shadow

Creates a shadow effect around an element's box.

<ClassList
  classData={[
    {
      className: 'bs',
      properties: [
        'box-shadow: 1px 3px 5px rgba(0, 0, 0, 0.1);'
      ]
    },
    {
      className: 'bs-xsm',
      properties: [
        'box-shadow: 1px 3px 5px -3px rgba(0,0,0,0.1);'
      ]
    },
    {
      className: 'bs-sm',
      properties: [
        'box-shadow: 1px 3px 5px -2px rgba(0,0,0,0.1);'
      ]
    },
    {
      className: 'bs-md',
      properties: [
        'box-shadow: 1px 3px 5px -1px rgba(0,0,0,0.1);'
      ]
    },
    {
      className: 'bs-lg',
      properties: [
        'box-shadow: 1px 3px 5px 1px rgba(0,0,0,0.1);'
      ]
    },
    {
      className: 'bs-xlg',
      properties: [
        'box-shadow: 1px 3px 5px 2px rgba(0,0,0,0.1);'
      ]
    },
  ]}
/>
