# Tabs

Compact elements allow easy switching between different sections or views, optimising user interface exploration

## Tab

Use these tabs to organize and present different sections or views within a single page or container, allowing users to switch between content categories or browse through related information effortlessly, improving content discoverability and user satisfaction.
