## Installation

### Install via NPM

```bash
npm i yummacss
```

```html
<link rel="stylesheet" href="node_modules/yummacss/dist/yumma.css">
```

---

### Integrate via CDN

#### Standard

Use this option for improved readability.

```html
<link href="https://cdn.jsdelivr.net/npm/yummacss/dist/yumma.css" rel="stylesheet" crossorigin="anonymous">
```

#### Minified

Use this option for faster loading.

```html
<link href="https://cdn.jsdelivr.net/npm/yummacss/dist/yumma.min.css" rel="stylesheet" crossorigin="anonymous">
```

## API

To display the Yumma CSS version in your console, import the following code snippet and call the `showVer` function:

```js
const { showVer } = require('./node_modules/yummacss/index');
```
