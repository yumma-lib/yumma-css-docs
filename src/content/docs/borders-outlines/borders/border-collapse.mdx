Controls the collapsing or separation of table borders.

<Utility
  data={[
    {
      className: 'bc-c',
      properties: [
        'border-collapse: collapse;'
      ]
    },
    {
      className: 'bc-s',
      properties: [
        'border-collapse: separate;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:bc-*`, `md:bc-*`, `lg:bc-*` enables the targeting of specific utilities in disparate viewports.
