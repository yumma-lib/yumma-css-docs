Controls the font weight of an element.

<Utility
  classPrefix="fw-"
  excludeZero
  incrementPrefix={100}
  incrementValue={100}
  propNames={["font-weight"]}
  range={9}
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:fw-*`, `md:fw-*`, `lg:fw-*`, and `xxl:fw-*` allows targeting specific utilities in different viewports.

```html "md:fw-600"
<div class="fw-400 md:fw-600 ..."></div>
```

### Using variants

Using `:hover` variants such as `h:fw-*` allows you to override elements and change their values when hovering over them.

```html "h:fw-600"
<div class="fw-400 h:fw-600 ..."></div>
```
