Controls the font style of an element.

<Utility
  data={[
    {
      className: 'fs-i',
      properties: [
        'font-style: italic;'
      ]
    },
    {
      className: 'fs-n',
      properties: [
        'font-style: normal;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:fs-*`, `md:fs-*`, `lg:fs-*` enables the targeting of specific utilities in disparate viewports.
