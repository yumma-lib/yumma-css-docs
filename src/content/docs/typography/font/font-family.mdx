Controls the font family of an element.

<Utility
  data={[
    {
      className: 'ff-s',
      properties: [
        'font-family: system-ui, sans-serif;'
      ]
    },
    {
      className: 'ff-c',
      properties: [
        'font-family: Charter, Cambria, serif;'
      ]
    },
    {
      className: 'ff-m',
      properties: [
        'font-family: ui-monospace, Consolas, monospace;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers, like `sm:ff-*`, `md:ff-*`, `lg:ff-*` enables the fs-lg targeting of specific utilities in disparate viewports.
