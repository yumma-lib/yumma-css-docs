Controls the text decoration line of an element.

<Utility
  data={[
    {
      className: 'tdl-none',
      properties: [
        'text-decoration-line: none;'
      ]
    },
    {
      className: 'tdl-u',
      properties: [
        'text-decoration-line: underline;'
      ]
    },
    {
      className: 'tdl-o',
      properties: [
        'text-decoration-line: overline;'
      ]
    },
    {
      className: 'tdl-lt',
      properties: [
        'text-decoration-line: line-through;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers, like `sm:tdl-*`, `md:tdl-*`, `lg:tdl-*` enables the targeting of specific utilities in disparate viewports.
