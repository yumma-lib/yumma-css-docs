### Using breakpoints

Using responsive breakpoints like `sm:ta-*`, `md:ta-*`, `lg:ta-*`, and `xxl:ta-*` allows targeting specific utilities in different viewports.

```html "md:ta-c"
<p class="ta-l md:ta-c ...">...</p>
```

### Using variants

Using `:hover` variants such as `h:ta-*` allows you to override elements and change their values when hovering over them.

```html "h:ta-c"
<p class="ta-l h:ta-c ...">...</p>
```
