Controls the style of a list.

<Utility
  data={[
    {
        className: 'lst-d',
        properties: [
            'list-style-type: disc;'
        ]
    },
    {
        className: 'lst-c',
        properties: [
            'list-style-type: circle;'
        ]
    },
    {
        className: 'lst-s',
        properties: [
            'list-style-type: square;'
        ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:lst-*`, `md:lst-*`, `lg:lst-*` enables the targeting of specific utilities in disparate viewports.
