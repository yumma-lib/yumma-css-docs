Controls the caption element orientation in a table.

<Utility
  data={[
    {
      className: 'cs-b',
      properties: [
        'caption-side: bottom;'
      ]
    },
    {
      className: 'cs-t',
      properties: [
        'caption-side: top;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:cs-*`, `md:cs-*`, `lg:cs-*` enables the targeting of specific utilities in disparate viewports.
