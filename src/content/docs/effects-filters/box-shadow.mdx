Controls the box shadow of an element.

<Utility
  data={[
    {
      className: "bs-none",
      properties: ["box-shadow: none;"],
    },
    {
      className: "bs-xs",
      properties: ["box-shadow: 1px 3px 5px -3px rgba(0,0,0,0.1);"],
    },
    {
      className: "bs-sm",
      properties: ["box-shadow: 1px 3px 5px -2px rgba(0,0,0,0.1);"],
    },
    {
      className: "bs-md",
      properties: ["box-shadow: 1px 3px 5px -1px rgba(0,0,0,0.1);"],
    },
    {
      className: "bs-lg",
      properties: ["box-shadow: 1px 3px 5px 1px rgba(0,0,0,0.1);"],
    },
    {
      className: "bs-xl",
      properties: ["box-shadow: 1px 3px 5px 2px rgba(0,0,0,0.1);"],
    },
  ]}
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:bs-*`, `md:bs-*`, `lg:bs-*`, and `xxl:bs-*` allows targeting specific utilities in different viewports.

```html "md:bs-md"
<div class="bs-sm md:bs-md ..."></div>
```

### Using variants

Using `:hover` variants such as `h:bs-*` allows you to override elements and change their values when hovering over them.

```html "h:bs-md"
<div class="bs-sm h:bs-md ..."></div>
```
