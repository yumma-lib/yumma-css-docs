Controls the aspect ratio of an element.

<Utility
  data={[
    {
      className: 'ar-s',
      properties: [
        'aspect-ratio: 1/1;'
      ]
    },
    {
      className: 'ar-l',
      properties: [
        'aspect-ratio: 16/9;'
      ]
    },
    {
      className: 'ar-auto',
      properties: [
        'aspect-ratio: auto;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers, like `sm:ar-*`, `md:ar-*`, `lg:ar-*` enables the targeting of specific utilities in disparate viewports.
