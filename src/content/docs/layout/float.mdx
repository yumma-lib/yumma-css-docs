Controls the content wrapping around an element.

<Utility
  data={[
    {
      className: 'flo-none',
      properties: [
        'float: none;'
      ]
    },
    {
      className: 'flo-l',
      properties: [
        'float: left;'
      ]
    },
    {
      className: 'flo-r',
      properties: [
        'float: right;'
      ]
    },
    {
      className: 'flo-is',
      properties: [
        'float: inline-start;'
      ]
    },
    {
      className: 'flo-ie',
      properties: [
        'float: inline-end;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers, like `sm:flo-*`, `md:flo-*`, `lg:flo-*` enables the targeting of specific utilities in disparate viewports.
