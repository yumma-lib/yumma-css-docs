Controls the display box type of an element.

<Utility
  data={[
    {
      className: "d-b",
      properties: ["display: block;"],
    },
    {
      className: "d-i",
      properties: ["display: inline;"],
    },
    {
      className: "d-ib",
      properties: ["display: inline-block;"],
    },
    {
      className: "d-f",
      properties: ["display: flex;"],
    },
    {
      className: "d-if",
      properties: ["display: inline-flex;"],
    },
    {
      className: "d-g",
      properties: ["display: grid;"],
    },
    {
      className: "d-fr",
      properties: ["display: flow-root;"],
    },
    {
      className: "d-none",
      properties: ["display: none;"],
    },
  ]}
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:d-*`, `md:d-*`, `lg:d-*`, and `xxl:d-*` allows targeting specific utilities in different viewports.

```html "md:d-b"
<div class="d-none md:d-b ..."></div>
```

### Using variants

Using `:hover` variants such as `h:d-*` allows you to override elements and change their values when hovering over them.

```html "h:d-b"
<div class="d-none h:d-b ..."></div>
```
