### Using breakpoints

Using responsive breakpoints like `sm:ac-*`, `md:ac-*`, `lg:ac-*`, and `xxl:ac-*` allows targeting specific utilities in different viewports.

```html "md:ac-e"
<div class="ac-s md:ac-e ac-"></div>
```

### Using variants

Using `:hover` variants such as `h:ac-*` allows you to override elements and change their values when hovering over them.

```html "h:ac-e"
<div class="ac-s h:ac-e ac-"></div>
```
