Controls how flex items are wrapped.

<Utility
  data={[
    {
      className: "fw-nw",
      properties: ["flex-wrap: nowrap;"],
    },
    {
      className: "fw-w",
      properties: ["flex-wrap: wrap;"],
    },
    {
      className: "fw-wr",
      properties: ["flex-wrap: wrap-reverse;"],
    },
  ]}
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:fw-*`, `md:fw-*`, `lg:fw-*`, and `xxl:fw-*` allows targeting specific utilities in different viewports.

```html "md:fw-w"
<div class="fw-wr md:fw-w ..."></div>
```

### Using variants

Using `:hover` variants such as `h:fw-*` allows you to override elements and change their values when hovering over them.

```html "h:fw-w"
<div class="fw-wr h:fw-w ..."></div>
```
