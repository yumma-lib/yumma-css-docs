Controls the size of grid columns created implicitly.

<Utility
  data={[
    {
      className: "gac-min",
      properties: ["grid-auto-columns: min-content;"],
    },
    {
      className: "gac-max",
      properties: ["grid-auto-columns: max-content;"],
    },
    {
      className: "gac-auto",
      properties: ["grid-auto-columns: auto;"],
    },
  ]}
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:gac-*`, `md:gac-*`, `lg:gac-*`, and `xxl:gac-*` allows targeting specific utilities in different viewports.

```html "md:gac-auto"
<div class="gac-min md:gac-auto ..."></div>
```

### Using variants

Using `:hover` variants such as `h:gac-*` allows you to override elements and change their values when hovering over them.

```html "h:gac-auto"
<div class="gac-min h:gac-auto ..."></div>
```
