Controls the size of grid lines created implicitly.

<Utility
  data={[
    {
        className: 'gar-min',
        properties: [
            'grid-auto-rows: min-content;'
        ]
    },
    {
        className: 'gar-max',
        properties: [
            'grid-auto-rows: max-content;'
        ]
    },
    {
        className: 'gar-auto',
        properties: [
            'grid-auto-rows: auto;'
        ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:gar-*`, `md:gar-*`, `lg:gar-*` enables the targeting of specific utilities in disparate viewports.
