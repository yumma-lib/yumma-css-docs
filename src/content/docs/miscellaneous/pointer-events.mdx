  data={[
    {

        properties: [

        ]
    },
    {
        className: 'pe-none',
        properties: [
            'pointer-events: none;'
        ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:pe-*`, `md:pe-*`, `lg:pe-*` enables the targeting of specific utilities in disparate viewports.
