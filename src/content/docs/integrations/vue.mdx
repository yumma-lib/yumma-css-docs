<Steps>
    1. Install Yumma CSS in your Vue application:

        ```bash title="Installing dependencies..."
        npm install yummacss@latest
        ```

    2. Import Yumma CSS in your main JavaScript or CSS file:

        <Code code={onCSS} lang="javascript" title="main.js" />

    3. Start your Vue development server:

        ```bash title="Installing dependencies..."
        npm run dev
        ```

    4. Begin using Yumma CSS in your Vue project:

        ```vue title="src/App.vue" {2-4}
        <template>
          <div class="ai-c bg-blue d-f jc-c h-1/1">
            <h1 class="fs-lg fw-700 tc-white">Yumma CSS + Vue</h1>
          </div>
        </template>

        <script>
        export default {
          name: 'App',
        }
        </script>

        <style>
        @import 'yummacss/dist/yumma.css';
        </style>
        ```

</Steps>
