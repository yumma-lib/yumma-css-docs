export const onHTML = `<!-- Minified version -->
<link href="/lib/yummacss/dist/yumma.min.css" rel="stylesheet" crossorigin="anonymous" />`;

<Steps>

    1. **Install Yumma CSS using LibMan:**

        Navigate to the solution your project:

        ```bash title="Terminal"
        cd project-name\solution-name
        ```

        Install the Yumma CSS Library using [LibMan](https://learn.microsoft.com/en-us/aspnet/core/client-side/libman/libman-cli?view=aspnetcore-8.0)

        ```bash title="Installing dependencies..."
        libman install yummacss@latest --provider jsdelivr --destination lib/yummacss
        ```

    3. **Create a new page in your web app directory.**

        Once you've installed it, you should see a new `lib` folder in your project directory.

        <FileTree>
            - **Index.aspx**
                - Index.aspx.cs
                - Index.aspx.designer.cs
                - lib
                    - yummacss/
                - libman.json
                - packages.config
                - Properties
                - AssemblyInfo.cs
                - Web.config
                - Web.Debug.config
                - Web.Release.config
        </FileTree>

    2. **Include Yumma CSS in your ASPX page in your `<head>` section:**

        <Code code={onHTML} lang="html" title="Index.aspx" />

    3. **Start your .NET development server:**

        You can also press the `F5` key to access the development server.

    4. **Begin using Yumma CSS in your .NET project:**

        Now you can use Yumma CSS Utility Classes inside markup.

        ```html title="Index.aspx"
        <form id="form1" runat="server">
            <div class="ai-c bg-d-lead-4 d-f jc-c h-1/1">
                <h1 class="fs-lg fw-700 tc-white">Yumma CSS + .NET</h1>
            </div>
        </form>
        ```

</Steps>
