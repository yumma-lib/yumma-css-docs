Controls the spacing of child elements.

## Spacing X

Controls the left and right X-axis margins of child elements.

<Utility 
  classPrefix='s-x-' 
  classValues={['margin-left', 'margin-right']}
  range={100} 
  increment={0.25} 
  unit='rem'
/>

## Spacing Y

Controls the top and bottom Y-axis margins of child elements.

<Utility 
  classPrefix='s-y-' 
  classValues={['margin-top', 'margin-bottom']} 
  range={100} 
  increment={0.25} 
  unit='rem' 
/>

---

### Using breakpoints

Using responsive breakpoints like `sm:s-x-*`, `md:s-x-*`, `lg:s-x-*`, and `xxl:s-x-*` allows targeting specific utilities in different viewports.

```html "md:s-x-2"
<div class="s-x-1 md:s-x-2 ..."></div>
```

### Using variants

Using `:hover` variants such as `h:s-x-*` allows you to override elements and change their values when hovering over them.

```html "h:s-x-2"
<div class="s-x-1 h:s-x-2 ..."></div>
```
