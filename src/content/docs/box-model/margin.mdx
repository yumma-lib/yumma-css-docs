### Using breakpoints

Using responsive breakpoints like `sm:m-*`, `md:m-*`, `lg:m-*`, and `xxl:m-*` allows targeting specific utilities in different viewports.

```html "md:m-2"
<div class="m-1 md:m-2 ..."></div>
```

### Using variants

Using `:hover` variants such as `h:m-*` allows you to override elements and change their values when hovering over them.

```html "h:m-2"
<div class="m-1 h:m-2 ..."></div>
```
