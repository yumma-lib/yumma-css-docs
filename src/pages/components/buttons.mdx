# Buttons

Interactive elements designed to act and encourage greater user engagement.

### Button

<Sandpack
  theme="auto"
  files={{
    "/App.js": {
code: `import React from "react";
import "yummacss/dist/yumma.css";

const App = () => {
   return (
      <div className="ins h-1/1">
         <button class="btn-black t-white">Button</button>
      </div>
   );
}

export default App;
`,
},
  }}
customSetup={{ dependencies: { "yummacss": "latest" }}}
 template="react"
 options={{ editorHeight: 450}}
/>

### Complement

:::warning Complement Buttons are Deprecated!
Complement buttons are no longer available in the latest versions. If you want to use complement buttons, consider using Yumma CSS versions before to v1.1.0.
:::

<Sandpack
  theme="auto"
  files={{
    "/App.js": {
code: `import React from "react";
import "yummacss/dist/yumma.css";
