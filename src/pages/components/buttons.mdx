# Buttons

Interactive elements for user actions.

### Button

theme="auto"

const App = () => {

### Complement

:::warning Complement Buttons are Deprecated!
Complement buttons are no longer available in the latest versions. If you want to use complement buttons, consider using Yumma CSS versions before to v1.1.0.
:::
