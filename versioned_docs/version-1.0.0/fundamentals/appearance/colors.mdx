## Base colors

The foundations of our color palette are defined in Scss variables. Here's how the colors are defined:

```scss
$base-colors: (
    "red": rgb(242, 109, 80),
    "orange": rgb(246, 142, 85),
    "yellow": rgb(255, 244, 103),
    "green": rgb(59, 184, 120),
    "teal": rgb(26, 187, 180),
    "aqua": rgb(0, 191, 243),
    "blue": rgb(67, 140, 202),
    "violet": rgb(133, 95, 168),
    "magenta": rgb(168, 99, 168),
    "pink": rgb(240, 110, 169),
    "gray": rgb(153, 153, 153),
    "lead": rgb(63, 63, 78),
    "black": black,
    "white": white
) !default;
```
