The `cnn{}` class provides a container with a responsive width, making it suitable for various screen sizes.

<ClassList
  classData={[
    {
      className: 'cnn',
      properties: [
        'width: 100%;',
        'max-width: 1200px;',
        'margin: 0 auto;',
        'padding: 0 20px;',
        'box-sizing: border-box;'
      ]
    },
  ]}
/>

```html
<div class="cnn">
  <!-- ... -->
</div>
```
