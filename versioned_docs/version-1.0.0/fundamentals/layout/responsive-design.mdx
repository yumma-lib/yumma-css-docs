## Breakpoints

Responsive breakpoints allow you to create flexible and adaptable layouts for your web projects. These breakpoints provide fine-grained control over your layout's behavior on different devices.

:::info Using Breakpoints
Include a [`row{}`](#row) class in the main container `<div>`. This ensures proper alignment and spacing of your columns. Here's a use case example:

```html
<div class="row">
    <!-- highlight-start -->
    <div class="col-*-[breakpoint]">
    <!-- highlight-end -->
        <!-- ... -->
    </div>
</div>
```
:::

Breakpoints are defined as Scss variables, here's how:

```scss
$breakpoints: (
    "xsm": 0,
    "sm": 480px,
    "md": 720px,
    "lg": 960px,
    "xlg": 1200px
);
```
