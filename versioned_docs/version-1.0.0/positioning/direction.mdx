# Direction

Controls the direction of the flexible items within a container.

<ClassList
  classData={[
    {
      className: 'dir-b',
      properties: [
        'bottom: 0rem;'
      ]
    },
    {
      className: 'dir-i',
      properties: [
        'inset: 0rem;'
      ]
    },
    {
      className: 'dir-l',
      properties: [
        'left: 0rem;'
      ]
    },
    {
      className: 'dir-r',
      properties: [
        'right: 0rem;'
      ]
    },
    {
      className: 'dir-t',
      properties: [
        'top: 0rem;'
      ]
    },
  ]}
/>
