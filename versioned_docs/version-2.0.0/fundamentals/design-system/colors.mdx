## Default colors

The foundations of our color palette are defined in Scss variables. Here's how the colors are defined:

```scss
// variables
$yma-colors-red: rgba(215, 61, 61);
$yma-colors-orange: rgba(224, 104, 20);
$yma-colors-yellow: rgba(211, 161, 7);
$yma-colors-green: rgba(31, 177, 85);
$yma-colors-teal: rgba(18, 166, 149);
$yma-colors-cyan: rgba(5, 164, 191);
$yma-colors-blue: rgba(53, 117, 221);
$yma-colors-indigo: rgba(89, 92, 217);
$yma-colors-violet: rgba(125, 83, 221);
$yma-colors-pink: rgba(212, 65, 138);
$yma-colors-gray: rgba(96, 103, 115);
$yma-colors-lead: rgba(63, 63, 78);
$yma-colors-black: rgba(0, 0, 0);
$yma-colors-white: rgba(255, 255, 255);
$yma-colors-transparent: rgba(255, 255, 255, 0);
```
