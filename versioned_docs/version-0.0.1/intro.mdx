# Introduction

Yumma CSS, created by [Renildo Pereira](https://rrenildopereiraa.github.io/renildo/), is a multipurpose CSS library that offers a diverse range of utility classes and components. These components include buttons, navigation bars, cards, and more, making UI development more straightforward and streamlined. Yumma CSS provides a comprehensive set of tools and functionalities to simplify development processes.

## Installation

Start building websites with Yumma CSS the way you want.

### Install via npm

Installing via [package manager](https://www.npmjs.com/package/yummacss):

:::warning No longer available from the package manager!
This version is outdated and is exclusively accessible through a content delivery network. For improved support and access to the latest features, we recommend installing the [latest versions](/docs/intro).
:::

<br/>

### Include via cdn

Including using [content delivery network](https://www.jsdelivr.com/package/npm/yummacss):

#### Standard:

```html
<link href="https://cdn.jsdelivr.net/npm/yummacss/dist/yumma.css" rel="stylesheet" crossorigin="anonymous">
```
