A set of default styles designed for consistency in all browsers.

## Universal selector

Ensures consistent box sizing behavior across all elements.

```scss
* {
    color: inherit;
    font-family: 'Poppins', Arial, sans-serif; // uses google font api
    margin: 0;   
}
```

## Link decoration
Removes default text decoration from links for cleaner appearance.

```scss
a {
    text-decoration: none;
}
```

## Horizontal rule
Applies a subtle border to horizontal rules for visual separation.

```scss
hr {
    border: 0;
    border-top: 1px dotted #efefef;
}
```

## Smooth Scrolling
Applies smooth scrolling behavior to HTML elements.

```scss
html {
    scroll-behavior: smooth;
}
```

## Responsive Images
Ensures images resize appropriately within their containers.

```scss
img {
    max-width: 100%;
}
```

## List normalization
Removes default padding and list-style from unordered lists.

```scss
ul {
    padding: 0;
    list-style-type: none;
}
```
