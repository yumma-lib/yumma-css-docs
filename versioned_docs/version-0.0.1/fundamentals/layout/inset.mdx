# Inset 

The `ins{}` class is designed to center content both horizontally and vertically within its parent container. It uses flexbox properties to achieve this, making it easy to create centered elements without the need for complex CSS.

<ClassList
  classData={[
    {
      className: 'ins',
      properties: [
        'align-items: center;',
        'display: flex;',
        'justify-content: center;',
      ]
    },
  ]}
/>

```html
<div class="ins">
  <!-- ... -->
</div>
```
