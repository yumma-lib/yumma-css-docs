# Get started with Yumma CSS

Yumma CSS provides a set of utility classes and components to help you ship and build applications faster, whether you're designing static pages or dynamic components.

No need to set anything up, just install and off you go!

## Installation

Start building websites with Yumma CSS the way you want.

### Install via npm

Installing via [package manager](https://www.npmjs.com/package/yummacss):

:::warning No longer available from the package manager!
This version is outdated and is exclusively accessible through a content delivery network. For improved support and access to the latest features, we recommend installing the [latest versions](/docs/installation).
:::

<br/>

### Include via cdn

Including using [content delivery network](https://www.jsdelivr.com/package/npm/yummacss):

#### Standard

```html
<link href="https://cdn.jsdelivr.net/npm/yummacss/dist/yumma.css" rel="stylesheet" crossorigin="anonymous">
```
