# Column

Defines how many columns an element should be divided into and their width.

:::info Responsive Column Classes
For instance, `col-xsm` is for extra-small screens, `col-sm` for small screens, `col-md` for medium screens, `col-lg` for large screens, and `col-xlg` for extra-large screens. This flexibility enables easy layout adjustments for various devices, eliminating the need for separate CSS for each screen size.
:::
