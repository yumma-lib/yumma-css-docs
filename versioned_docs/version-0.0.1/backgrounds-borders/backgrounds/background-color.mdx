### Customizing colors

Use `bg-l-[color]-*` for a lighter color and `bg-d-[color]-*` for a darker color.

<ColorPalette
  colorData={[
    {
      name: 'Pink',
      value: '#f06ea9'
    }
  ]}
/>

### Using hover states

Use `bg-h-*` to modify elements to override other background classes.

  codeData={`<div class="bg-gray h-8 bg-h-pink rad-1 w-8"></div>`}
