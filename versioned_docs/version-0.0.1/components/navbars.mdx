# Navbars

Top horizontal bars provide seamless navigation for an enhanced user experience

## Default Navbar

Use these navbars to create a clear and easily accessible menu at the top of your website or application, allowing users to navigate through different sections, explore content, and reach important pages with ease.

:::caution Customizing the Default Navbar
This is a pre-built navbar component and certain styles, such as background color, cannot be changed. These styles are intended to maintain a consistent and cohesive design. For extensive customization, consider creating a custom navbar from scratch, or explore our other navbar components.
:::
