# Row Gap

Sets the gap between rows in a grid layout.

:::info Creating Grid Layouts with Row Gaps
Specify a container element and apply these classes to create a grid layout with personalized row spacing:

- `cg-{size}`: Control column spacing.
- `rg-{size}`: Control row spacing.
- `gtc-{size}`: Specify grid columns.
- `gtr-{size}`: Specify grid rows.
- `d-g-{size}`: Designate as a grid container.
:::
