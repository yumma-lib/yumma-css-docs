# Row

The `row` class is designed to create flexible and responsive row layouts within your web projects. It utilizes CSS Flexbox properties to create rows that automatically wrap when content exceeds the available space.

<ClassList
  classData={[
    {
      className: 'row',
      properties: [
        'display: flex;',
        'flex-flow: row wrap;',
      ]
    },
  ]}
/>

### Creating Responsive Rows

The `row` class simplifies the process of creating responsive row layouts. It allows you to place elements side by side in a row while ensuring that they automatically wrap to the next line when the screen width is insufficient.
