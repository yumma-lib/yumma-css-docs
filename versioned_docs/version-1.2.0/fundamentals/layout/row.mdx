# Row

The `row{}` class simplifies the process of creating responsive row layouts. It allows you to place elements side by side in a row while ensuring that they automatically wrap to the next line when the screen width is insufficient.

<ClassList
  classData={[
    {
      className: "row",
      properties: ["display: flex;", "flex-flow: row wrap;"],
    },
  ]}
/>

### Basic example

```html
<!-- highlight-start -->
<div class="row">
<!-- highlight-end -->
  <div class="[xs | sm | md | lg | xl]:col-[value]">
    <!-- ... -->
  </div>
</div>
```
