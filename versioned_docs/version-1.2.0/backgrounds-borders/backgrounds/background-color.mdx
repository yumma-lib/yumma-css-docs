### Customizing colors

Use `bg-l-[color]-*` for a lighter color and `bg-d-[color]-*` for a darker color.

```html
<!-- light variations -->
<div class="bg-l-pink-6"></div>
<div class="bg-l-pink-5"></div>
<div class="bg-l-pink-4"></div>
<div class="bg-l-pink-3"></div>
<div class="bg-l-pink-2"></div>
<div class="bg-l-pink-1"></div>

<!-- base color -->
<div class="bg-pink"></div>

<!-- dark variations -->
<div class="bg-d-pink-1"></div>
<div class="bg-d-pink-2"></div>
<div class="bg-d-pink-3"></div>
<div class="bg-d-pink-4"></div>
<div class="bg-d-pink-5"></div>
<div class="bg-d-pink-6"></div>
```
