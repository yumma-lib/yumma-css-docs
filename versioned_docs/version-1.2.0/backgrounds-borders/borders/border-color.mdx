### Customizing colors

Use `bc-l-[color]-*` for a lighter color and `bc-d-[color]-*` for a darker color.

```html
<!-- light variations -->
<div class="bc-l-pink-6"></div>
<div class="bc-l-pink-5"></div>
<div class="bc-l-pink-4"></div>
<div class="bc-l-pink-3"></div>
<div class="bc-l-pink-2"></div>
<div class="bc-l-pink-1"></div>

<!-- base color -->
<div class="bc-pink"></div>

<!-- dark variations -->
<div class="bc-d-pink-1"></div>
<div class="bc-d-pink-2"></div>
<div class="bc-d-pink-3"></div>
<div class="bc-d-pink-4"></div>
<div class="bc-d-pink-5"></div>
<div class="bc-d-pink-6"></div>
```
