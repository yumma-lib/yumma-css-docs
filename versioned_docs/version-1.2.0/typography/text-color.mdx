## Color variations

You have access to 6 shades of both dark and light colors. This means you can use classes like `t-l-pink-1` through `t-l-{color}-6` for light text colors and `t-d-pink-1` through `t-d-{color}-6` for dark text colors.

### Using hover states

Using different hover states to modify different elements by using `h:t-gray`, for example, you will override the existing utility class `t-gray` on hover.

<ClassPreview
  codeData={`
    <h3 class="t-gray h:t-pink">The quick brown fox jumps over the lazy dog.</h3>
  `}
  isPadded
  isCentered
  isVersion = '1.2.0'
/>

```html
<h3 class="t-gray h:t-pink">The quick brown fox jumps over the lazy dog.</h3>
```
