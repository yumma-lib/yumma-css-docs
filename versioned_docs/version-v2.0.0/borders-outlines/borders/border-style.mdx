  classData={[
    {
      className: 'b-none',
      properties: [
        'border-style: none;'
      ]
    },
    {
      className: 'b-d',
      properties: [
        'border-style: dashed;'
      ]
    },
    {
      className: 'b-h',
      properties: [
        'border-style: hidden;'

    },
    {
      className: 'b-s',

        'border-style: solid;'
      ]
    },
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:b-*`, `md:b-*`, `lg:b-*` enables the targeting of specific utilities in disparate viewports.
