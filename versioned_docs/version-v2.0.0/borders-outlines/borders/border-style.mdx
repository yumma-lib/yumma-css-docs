Controls the style of the borders of an element.

<Utility
  data={[
    {
      className: 'b-none',
      properties: [
        'border-style: none;'
      ]
    },
    {
      className: 'b-d',
      properties: [
        'border-style: dashed;'
      ]
    },
    {
      className: 'b-h',
      properties: [
        'border-style: hidden;'
      ]
    },
    {
      className: 'b-s',
      properties: [
        'border-style: solid;'
      ]
    }
  ]}
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:b-*`, `md:b-*`, `lg:b-*` enables the targeting of specific utilities in disparate viewports.
