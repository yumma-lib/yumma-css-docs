## Default colors

The foundations of our color palette are defined in Scss variables. Here's how the colors are defined:

```scss
// variables
$yma-colors-red: rgb(215, 61, 61);
$yma-colors-orange: rgb(224, 104, 20);
$yma-colors-yellow: rgb(211, 161, 7);
$yma-colors-green: rgb(31, 177, 85);
$yma-colors-teal: rgb(18, 166, 149);
$yma-colors-cyan: rgb(5, 164, 191);
$yma-colors-blue: rgb(53, 117, 221);
$yma-colors-indigo: rgb(89, 92, 217);
$yma-colors-violet: rgb(125, 83, 221);
$yma-colors-pink: rgb(212, 65, 138);
$yma-colors-gray: rgb(96, 103, 115);
$yma-colors-slate: rgb(123, 134, 151);
$yma-colors-lead: rgb(63, 63, 78);
$yma-colors-black: black;
$yma-colors-white: white;
$yma-colors-silver: rgb(191, 194, 199);
$yma-colors-transparent: transparent;
```
