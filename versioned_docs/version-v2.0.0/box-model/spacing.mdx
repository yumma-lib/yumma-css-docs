Controls the spacing of child elements.

## Spacing X

Controls the left and right X-axis margins of child elements.

<Utility 
  classPrefix='s-x-' 
  classValue={['margin-left', 'margin-right']}
  range={100} 
  increment={0.25} 
  unit='rem'
/>

## Spacing Y

Controls the top and bottom Y-axis margins of child elements.

<Utility 
  classPrefix='s-y-' 
  classValue={['margin-top', 'margin-bottom']} 
  range={100} 
  increment={0.25} 
  unit='rem' 
/>

### Using breakpoints

The utilization of breakpoint modifiers like `sm:s-x-*`, `md:s-x-*`, `lg:s-x-*` enables the targeting of specific utilities in disparate viewports.
