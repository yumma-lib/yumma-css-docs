# Get started with Yumma CSS

Yumma CSS provides a set of utility classes and components to help you ship and build applications faster, whether you're designing static pages or dynamic components.

No need to set anything up, just install and off you go!

## Installation

Start building websites with Yumma CSS the way you want:

<Tabs queryString="using">
  <TabItem value="npm" label="Via NPM">
    1. Install Yumma CSS via npm:

    ```bash
    npm install yummacss@2.0.0
    ```
    2. Include the compiled CSS file in your `<head>` element:

    ```html
    <!-- Standard version -->
    <link rel="stylesheet" href="node_modules/yummacss/dist/yumma.css">
    ```
  </TabItem>
  <TabItem value="cdn" label="Try the CDN">
    How to include Yumma CSS via cdn:

    ```html
    <!-- Standard version -->
    <link href="https://cdn.jsdelivr.net/gh/yumma-lib/yumma-css@2.0.0/dist/yumma.css" rel="stylesheet" crossorigin="anonymous">
    ```
  </TabItem>
</Tabs>

### CSS options

Once Yumma CSS has been installed via [package manager](https://www.npmjs.com/package/yummacss) or [cdn](https://www.jsdelivr.com/package/npm/yummacss), you will be able to access different compiled CSS files.

- `/dist/yumma.css` — Standard
- `/dist/yumma.min.css` — Minified
- `/dist/yumma-core.css` — Standard (without [Stylecent](/docs/stylecent))
- `/dist/yumma-core.min.css` — Minified (without [Stylecent](/docs/stylecent))
