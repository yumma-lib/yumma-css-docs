    },
    {

            'text-decoration: none;'
        ]
    }
  ]}

### Using breakpoints

The utilization of breakpoint modifiers like `sm:td-*`, `md:td-*`, `lg:td-*` enables the targeting of specific utilities in disparate viewports.
