# Text Decoration Line

Adds visual styling to text decorations.

<ClassList
  classData={[
    {
      className: 'tdl-none',
      properties: [
        'text-decoration-line: none;'
      ]
    },
    {
      className: 'tdl-u',
      properties: [
        'text-decoration-line: underline;'
      ]
    },
    {
      className: 'tdl-o',
      properties: [
        'text-decoration-line: overline;'
      ]
    },
    {
      className: 'tdl-lt',
      properties: [
        'text-decoration-line: line-through;'
      ]
    },
  ]}
/>
