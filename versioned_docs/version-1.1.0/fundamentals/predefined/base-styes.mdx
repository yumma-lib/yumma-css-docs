# Base Styles

Base Styles provides a consistent starting point for styling web pages.

## Universal selector

Ensures consistent box sizing behavior across all elements.

```scss
* {
    color: inherit;
    margin: 0;
    font-family: system-ui, sans-serif;
}

::before,
::after {
    box-sizing: border-box;
}
```

## Font Family Inheritance
Allows the body element to inherit the font family from its parent.

```scss
body {
    font-family: inherit;
}
```

## Link decoration
Removes default text decoration from links for cleaner appearance.

```scss
a {
    color: inherit;
    text-decoration: none;
}
```

## Horizontal rule
Applies a subtle border to horizontal rules for visual separation.

```scss
hr {
    border: 0;
    border-top: 1px solid #efefef;
    margin: 1em 0;
}
```

## Smooth Scrolling
Enables smooth scrolling behavior for HTML documents.

```scss
html {
    scroll-behavior: smooth;
}
```

## Media elements
Ensures images, videos, and other media elements are responsive by default.

```scss
img,
picture,
svg,
video {
    max-width: 100%;
    display: block;
}
```

## List normalization
Removes default padding and list-style from unordered lists.

```scss
ul {
    padding: 0;
    list-style: none;
}
```
