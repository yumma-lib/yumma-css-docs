# Dimension

Sets the height and the width of an element's content box.

<ClassList 
  baseClassName='dim-' 
  propertyNames={['height', 'width']} 
  range={100} 
  increment={0.25} 
  unit='rem' 
  additionalClasses={[
    {name: 'auto', value: 'auto'},
    {name: 'full', value: '100%'},
    {name: 'half', value: '50%'},
    {name: '1/1', value: '100vh'},
    {name: '1/2', value: '50vh'},
  ]} 
/>
