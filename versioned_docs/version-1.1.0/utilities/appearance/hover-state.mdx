# Hover State

Manages the appearance of elements when users hover over them, adding interactivity.

### Basic examples

#### Background Color

<CodePreview
  htmlContent={`
    <div class="bg-magenta h-6 h:bg-black p-1 rad-1 w-6"></div>
  `}
  isPadded = {true}
  isCentered = {true}
  isVersion = 'latest'
/>

<div class="bg-magenta h-6 h:bg-black p-1 rad-1 w-6"></div>
