## Default colors

The foundations of our color palette are defined in Scss variables. Here's how the colors are defined:

```scss
// map
$base-colors: (
    "red": #f26d50,
    "orange": #f68e55,
    "yellow": #fff467,
    "green": #3bb878,
    "teal": #1abbb4,
    "aqua": #00bff3,
    "blue": #438cca,
    "magenta": #855fa8,
    "violet": #a863a8,
    "pink": #f06ea9,
    "gray": #999999
    "black": black,
    "white": white
);
```
