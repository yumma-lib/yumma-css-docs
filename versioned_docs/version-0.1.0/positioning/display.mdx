# Display

Specifies the rendering behavior of an element.

<ClassList
  classData={[
    {
      className: 'dis-b',
      properties: [
        'display: block;'
      ]
    },
    {
      className: 'dis-f',
      properties: [
        'display: flex;'
      ]
    },
    {
      className: 'dis-g',
      properties: [
        'display: grid;'
      ]
    },
    {
      className: 'dis-i',
      properties: [
        'display: inline;'
      ]
    },
    {
      className: 'dis-ib',
      properties: [
        'display: inline-block;'
      ]
    },
    {
      className: 'dis-if',
      properties: [
        'display: inline-flex;'
      ]
    },
    {
      className: 'dis-none',
      properties: [
        'display: none;'
      ]
    },
  ]}
/>

```html
<div class="dis-b">
  <h2>Block-Level Heading</h2>
  <p>This is a block-level element. It takes up the full width of its parent container and starts on a new line.</p>
</div>
```
