### Customizing colors

Use `t-l-[color]-*` for a lighter color and `t-d-[color]-*` for a darker color.

```html
<!-- light variations -->
<div class="t-l-pink-6"></div>
<div class="t-l-pink-5"></div>
<div class="t-l-pink-4"></div>
<div class="t-l-pink-3"></div>
<div class="t-l-pink-2"></div>
<div class="t-l-pink-1"></div>

<!-- base color -->
<div class="t-pink"></div>

<!-- dark variations -->
<div class="t-d-pink-1"></div>
<div class="t-d-pink-2"></div>
<div class="t-d-pink-3"></div>
<div class="t-d-pink-4"></div>
<div class="t-d-pink-5"></div>
<div class="t-d-pink-6"></div>
```
