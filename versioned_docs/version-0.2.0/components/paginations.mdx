# Paginations

Navigation components that make it easy to browse through pages or sections

## Simple Pagination

Use these paginations to divide content into manageable sections or pages, enabling users to navigate through lengthy lists, articles, or search results effectively, enhancing user experience and accessibility.

  codeData={`

          </li>
          <li>
              <a class="bc-h-magenta bc-l-gray-4 bg-white p-1 rad-br-1 rad-tr-1 t-d-gray-5 t-h-magenta" href="#">Next</a>
