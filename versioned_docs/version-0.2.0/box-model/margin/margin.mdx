# Margin

Sets the margin around all four sides of an element.

<ClassList 
  baseClassName='m-' 
  propertyNames={['margin']} 
  range={100} 
  increment={0.75} 
  unit='rem' 
  additionalClasses={[
    {name: 'auto', value: 'auto'},
  ]} 
/>
