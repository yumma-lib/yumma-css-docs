# Width

Sets the width of an element's content box.

<ClassList 
  baseClassName='w-' 
  propertyName='width' 
  range={100} 
  increment={0.25} 
  unit='rem' 
  additionalClasses={[
    {name: 'auto', value: 'auto'},
    {name: 'full', value: '100%'},
    {name: 'half', value: '50%'}
  ]} 
/>
