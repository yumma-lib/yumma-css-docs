# Get started with Yumma CSS

Yumma CSS provides a set of utility classes and components to help you ship and build applications faster, whether you're designing static pages or dynamic components.

No need to set anything up, just install and off you go!

## Installation

Start building websites with Yumma CSS the way you want.

### Install via npm

Installing via [package manager](https://www.npmjs.com/package/yummacss):

```bash
npm i yummacss@0.1.1
```

Include the compiled CSS file in your `<head>` element

#### Standard

```html
<link rel="stylesheet" href="node_modules/yummacss/dist/yumma.css">
```
#### Minified

```html
<link rel="stylesheet" href="node_modules/yummacss/public/css/yumma.min.css">
```

<br/>

### Include via cdn

Including using [content delivery network](https://www.jsdelivr.com/package/npm/yummacss):

#### Standard:

```html
<link href="https://cdn.jsdelivr.net/npm/yummacss@0.1.1/dist/yumma.css" rel="stylesheet" crossorigin="anonymous">
```

#### Minified

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/yummacss@0.1.1/public/css/yumma.min.css" crossorigin="anonymous">
```
