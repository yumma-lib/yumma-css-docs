# Container

The `cnn` class provides a container with a responsive width, making it suitable for various screen sizes.

<ClassList
  classData={[
    {
      className: 'cnn',
      properties: [
        'width: 100%;',
        'max-width: 1200px;',
        'margin: 0 auto;',
        'padding: 0 20px;',
        'box-sizing: border-box;'
      ]
    },
  ]}
/>

### Responsive Container

The container class `cnn` is designed to create a responsive content container for your web projects. It ensures that your content looks well-organized and readable on different devices.

### Basic example

To use the `cnn` class, simply add it to your HTML elements. It will adapt to the available screen width while maintaining a maximum width of 1200 pixels. Here's an example:

```html
<div class="cnn">
  <!-- ... -->
</div>
```
